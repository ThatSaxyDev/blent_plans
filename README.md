# Nigerian Primary School Lesson Plans

A collection of lesson plans and lesson notes for Nigerian primary school teachers. Covers History and Social Studies subjects for Grades 3, 4, and 5.

## Structure

```
lessons/
├── history/
│   └── Grade X/
│       └── topic-folder/
│           ├── grade-x-history-topic-lesson-plan.md
│           └── grade-x-history-topic-lesson-notes.md
└── social studies/
    └── Grade X/
        └── topic-folder/
            ├── grade-x-social-studies-topic-lesson-plan.md
            └── grade-x-social-studies-topic-lesson-notes.md
```

## What's Included

Each lesson includes:

1. **Lesson Plan** - Formal record with:
   - Class, subject, duration, topic
   - Subtopics
   - Behavioral objectives
   - Reference materials
   - Previous knowledge
   - Presentation steps
   - Instructional materials
   - Evaluation questions

2. **Lesson Notes** - Teaching script with:
   - Lesson content
   - Detailed subtopic explanations
   - Simple language for primary school children
   - Classwork/assignment

## Agent Skill

This repo includes a custom Agent Skill for generating lesson plans using the `lesson-planner` skill.

To use: Configure the `.ite` folder with your Agent IDE to enable the lesson planner skill.

## Contributing

Add new lessons following the naming convention:
- Folder: `Grade X` (with space)
- Files: `grade-x-subject-topic-lesson-plan.md` and `grade-x-subject-topic-lesson-notes.md`
